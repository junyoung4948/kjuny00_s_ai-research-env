# Claude Code 전역 설정

이 파일은 `~/.claude/CLAUDE.md`에 복사되어 모든 프로젝트에 적용됩니다.

---

## 기본 행동 원칙

- 연구자는 컴퓨터 아키텍처/시스템 분야 대학원생입니다.
- 연구 프로젝트에서는 프로젝트 루트의 `CLAUDE.md`가 우선합니다.
- 이 전역 설정은 프로젝트 설정이 없을 때의 기본값입니다.

## 안전 규칙 (항상 적용)

- 파라미터 결정 시 한 번에 하나씩 확인 (Atomic Decision)
- 실험 결과 파일은 함부로 수정/삭제하지 않음
- 불확실하면 실행 전 연구자에게 확인
